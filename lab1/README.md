## 4. Как реализовано (Теория и фрагменты кода)

### 4.1. Семантическая структура и маршрутизация

**Теория:** Семантический HTML (Semantic HTML) вводит теги, которые четко описывают свое предназначение как для разработчика, так и для браузера (и вспомогательных технологий). Вместо безликих `<div id="header">` стандарт HTML5 предписывает использовать тег `<header>`.

Для реализации дополнительного задания была создана навигационная панель с использованием гиперссылок `<a>`. На каждой странице соответствующая ссылке страница выделяется классом `.active`, указывая пользователю его текущее местоположение.

**Фрагмент кода (Навигация в HTML):**

```html
<header>
    <div class="topnav">
        <a href="./home.html"><img src="./img/spacex-logo.svg" alt="SpaceX" /></a>
        <a href="./work_purpose.html">Цель работы</a>
        <a class="active" href="./calculator.html">Калькулятор</a>
        <a href="./about_author.html">Об авторе</a>

        <div class="theme-select">
            <label for="select_theme">Выбор темы: </label>
            <select id="select_theme" class="select-theme">
                <option value="dark">Темная</option>
                <option value="light">Светлая</option>
            </select>
        </div>
    </div>
</header>

```

### 4.2. Двумерная компоновка (CSS Grid)

**Теория:** Классический подход верстки блоками (`display: block`) требует создания множества узлов-оберток (строк) для формирования сетки элементов. CSS Grid Layout — это современная мощная система двумерной компоновки, позволяющая располагать дочерние элементы в колонки и строки прямо из родительского контейнера.

Для сетки пульта управления (калькулятора) был использован `display: grid`. Это позволило разместить все кнопки на одном уровне DOM-дерева, сделав HTML чище.

**Фрагмент кода (Реализация сетки в CSS):**

```css
/* Контейнер калькулятора */
.calc-grid {
    display: grid;
    /* Создание 6 равных колонок шириной по 50px каждая */
    grid-template-columns: repeat(6, 50px);
    gap: 5px; /* Стандартизированный отступ между ячейками */
    width: max-content;
}

/* Вывод результата растягивается на всю ширину сетки (от 1 линии до последней) */
.result {
    grid-column: 1 / -1;
    /* ... визуальные свойства ... */
}

/* Специальные кнопки растягиваются на 2 колонки */
.calc-btn.double-width {
    grid-column: span 2;
}

```

Для вывода результата вычислений в HTML был применен специализированный тег `<output id="result" class="result">0</output>`.

### 4.3. Каскадирование и переменные (CSS Custom Properties)

**Теория:** Использование «магических чисел» и жестко закодированных HEX-цветов по всему CSS-файлу приводит к проблемам масштабирования (нарушается принцип DRY — Don't Repeat Yourself). Нативные CSS-переменные позволяют определить палитру один раз в корневом селекторе `:root` и переиспользовать её. В отличие от препроцессоров (SASS/LESS), CSS-переменные существуют в рантайме браузера и могут быть изменены динамически, что необходимо для смены тем.

**Фрагмент кода (Палитра SpaceX в CSS):**

```css
:root {
    --mars-beige: #d7a384;
    --mars-brown: #b57756;
    --mars-grey: #503530;
    --mars-black: #2e242b;
    --black: #000000;
    --white: #ffffff;
    --white-35: rgba(255, 255, 255, 0.35);
    --black-50: rgba(0, 0, 0, 0.5);
}

/* Применение БЭМ-подобных модификаторов для стилизации состояний */
.calc-btn.primary {
    background-color: var(--mars-black);
}

.calc-btn.secondary {
    background-color: var(--mars-grey);
}

```

## 5. Вывод

В ходе выполнения лабораторной работы статический макет веб-интерфейса был успешно спроектирован и сверстан. Была разработана многостраничная архитектура (MPA).

Изучены и применены на практике современные технологии фронтенд-разработки:

* Семантические теги HTML5 улучшили структуру документа;
* Модель `CSS Grid` позволила создать строгую и легко масштабируемую сетку калькулятора без засорения DOM-дерева лишними `div`-контейнерами;

Интерфейс отвечает требованиям предметной области (стилистика марсианской миссии) и полностью готов к интеграции бизнес-логики на стороне клиента (Лабораторная работа №2).
